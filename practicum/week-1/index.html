In this practicum you will develop a responsive webpage using Bootstrap. The webpage that you need to develop is a simple verstion of the first page of this website: https://giftsear.ch

What you need to do:

  1. Create a github repository and name it mygiftsearch. Inside that repository create an html file and name it index.html. That file will contain your code for this practicum.
  2. Your mygiftsearch web page should have the following Boostrap components:
    1. Bootstrap container
    2. Bootstrap Image
    3. Bootstrap Button
    4. Bootstrap Card
    5. Bootstrap Navigation Bar
  3. You just need to develop one single page. The links in the navigation bar can be all broken links.
  4. Your design should be responsive, meaning it should look good on all devices.
  5. When you are done with your design, you need to publish your mygiftsearch repository. To do that do the followings:
    • Go to your mygiftsearch repository
    • Select the Settings tab
    • Navigate to Github Pages section
    • In the source select the master branch and then click the Save button
  6. Once you publish your webpage, it will be shown in the iframe below.
